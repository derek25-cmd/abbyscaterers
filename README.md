# abbyscaterers
Management system

**Features:**

*   Client Data Input: Intuitive form for entering and editing client details, including contact information, event preferences, and dietary restrictions.
*   Client List View: Display client information in a searchable and sortable table view.
*   Offline Data Storage: Securely store client data locally using the file system, without a database. App persists user changes without asking the user to manually save.
*   Recent Contact Indicator: Each client's information view shows when they were last contacted.
*   Data Export: Export client data to a common file format for external sharing or reporting.

**Client Parameters:**

*   Client ID
*   Company Name
*   Company email
*   Phone Number
*   Address 1
*   Address 2
*   Primary Location
*   Option to edit any client info.

**UI/UX:**

*   Grid-based layout for consistent spacing and alignment.
*   Clean, readable sans-serif font.
*   Simple, line-based icons for common actions and categories.
*   Subtle transitions and animations.
*   Colors: #800020, #F5F5DC, #B8860B
